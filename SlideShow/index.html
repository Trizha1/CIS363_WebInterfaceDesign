Description of the Process


The Keyframes Animation was used to automate the slideshow and to continually change the images.

The calculations I made to create the animations:
  • Total number of images to use in the slider
  • Length of animation for each image
  • Total duration of the animation
  • Calculate how many keyframes equals one second
